# Welcome to Kahf Browser: A Shariah Compliance Browser

## Cloning the Project

1. Clone the project with submodules:
   ```bash
   git clone --recurse-submodules git@github.com:Kahf-Browser/Android.git
    ```

2. If already cloned without submodules, run this command in the root directory. May take a while:
    ```bash
    git submodule update --init --recursive
    ```

3. Put the following files in the root directory of the project:
    - `posthog.properties`
    - `config.properties`
